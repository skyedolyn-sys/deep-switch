# Screenshots

Drop screenshots here when capturing them. Naming convention:

- `main-window.png` — main window showing provider list + sidebar
- `add-provider.png` — preset selector modal
- `provider-detail.png` — provider settings page
- `tray-menu.png` — macOS menu bar tray menu

Recommended dimensions: 1200×800 PNG, dark theme, Retina resolution.

Reference these from README.md via relative path:

```markdown
![Main window](./docs/screenshots/main-window.png)
```
